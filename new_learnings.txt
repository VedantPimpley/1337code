how to stringify and de-stringify a list of strings?        
        stringify:
        serialize into a single string, separated by a delimiter
        if delimiter exists in text, precede it with an escape character
        if escape character exists in text, precede it with itself

        destringify:
        go one character at a time
        hold normal characters in an accumulator
        if delimiter found, trigger word separation
        if escape character found, 
            treat the next character as a normal character and continue from the next-next character

right way to extract a digits from an integer is using modulo and floor division, not float division which is error prone.
    while n != 0:
        digits.add( n % 10 )
        n = n // 10

    format(n, f'0{l}b')
